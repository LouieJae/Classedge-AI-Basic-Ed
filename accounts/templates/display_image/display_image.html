Login Carousel Image Management

Display Status

Images marked as Displayed will appear in the login page carousel. You can toggle the display status by editing an image.

Note: If no images are marked for display, the system will use default images.

Image Guidelines

For best results in the carousel:

  • Use landscape orientation (16:9 ratio recommended)
  • Recommended size: 1200 × 675 pixels
  • Maximum file size: 2MB
  • Supported formats: JPG, PNG, WebP
